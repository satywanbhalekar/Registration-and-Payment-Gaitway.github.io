# registration-and-payment-form.
 In first step, request user details: username, email, password. In second step, request user for payment types: Credit card/debit card/Paypal/UPI. When user selects credit card or debit card, request card details from user and detect card type (VISA/ MasterCard etc.) on input and display it's icon near card number. When user selects paypal, request for valid ID.When user selects UPI, request user for a UPI ID.
